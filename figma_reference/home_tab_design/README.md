# 홈탭 디자인 참고 파일

Story Atlas 프론트엔드의 홈탭(메인 페이지) 디자인 조언을 위한 파일 모음입니다.

## 프로젝트 정보
- **프레임워크**: Next.js 16.0.1 (App Router)
- **스타일링**: Tailwind CSS 4 + CSS Modules
- **폰트**: Pretendard (메인), Dancing Script (필기체), KoPub 바탕체, 미생체, BM 을지로체

## 파일 설명

### 1. 핵심 페이지 및 레이아웃
- **page.js**: 홈 페이지 메인 컴포넌트
  - 카테고리별 콘텐츠 구성 (행사, 북카페, 브런치, 야외카페, 대형카페, 관광지)
  - API 데이터 패칭 로직
  - 상단 문구: "장소와 취향이 발견되는 공간"

- **layout.js**: 전체 레이아웃 및 헤더
  - 네비게이션 바 (로고, 탐색, 책갈피 만들기)
  - 폰트 설정
  - 전역 스타일 적용

### 2. 컴포넌트
- **CategoryCarousel.js**: 카테고리별 가로 스크롤 캐러셀
  - 드래그 스크롤 기능
  - 좌우 화살표 버튼
  - 카테고리 제목 + 설명 문구 + 별 아이콘 (외부 링크)

- **PlaceCard.js**: 장소 카드 UI (288x192 이미지 + 텍스트)
  - 이미지 + 장소명 + AI 요약 헤드라인
  - 호버 시 확대 효과

- **EventCard.js**: 행사 카드 UI (288x320 이미지 + 텍스트)
  - 세로형 포스터 이미지 + 행사명 + 헤드라인 + 기간
  - 종료된 행사 표시 (배지 + 반투명 처리)

### 3. 스타일
- **globals.css**: 전역 스타일 정의
  - 폰트 클래스 (.font-title, .font-pretendard, .font-dancing-script 등)
  - 색상 (완전한 검은색 #000000 강제 적용)
  - 애니메이션 (float, blob)

- **tailwind.config.mjs**: Tailwind 설정
  - 커스텀 폰트 정의
  - 플러그인 (scrollbar-hide)

### 4. 프로젝트 정보
- **package.json**: 의존성 및 프로젝트 메타데이터

## 현재 디자인 특징

### 레이아웃
- 최대 너비: 1152px (max-w-6xl)
- 좌우 패딩: 2rem (px-8)
- 헤더: sticky, 상단 고정

### 색상 팔레트
- 메인 텍스트: #000000 (완전한 검은색)
- 서브 텍스트: #6B7280 (gray-600)
- 배경: #FFFFFF (흰색)
- 호버/포커스: #2563EB (blue-600)

### 타이포그래피
- 제목: Pretendard Bold (700)
- 본문: Pretendard Medium (500)
- 로고: Pretendard ExtraBold (900), 소문자

### 카드 디자인
- 그림자: shadow-lg
- 둥근 모서리: 없음 (직각)
- 호버 효과: scale-105 (5% 확대)

### 캐러셀
- 카드 간격: 1rem (space-x-4)
- 카드 크기: 288px 고정 너비
- 스크롤: 가로 스크롤 (드래그 지원)
- 네비게이션: 좌우 화살표 버튼

## 디자인 개선 제안 요청 사항

1. **전체적인 비주얼 개선**
   - 현재 디자인이 너무 평면적이고 단조로운가요?
   - 색상 사용이 적절한가요? (거의 흑백)

2. **카드 디자인**
   - 그림자 효과가 적절한가요?
   - 둥근 모서리를 추가하는 것이 좋을까요?
   - 카드 크기와 비율이 적절한가요?

3. **타이포그래피**
   - 폰트 크기와 굵기가 적절한가요?
   - 가독성은 좋은가요?
   - 계층 구조가 명확한가요?

4. **레이아웃과 공간**
   - 여백이 적절한가요?
   - 콘텐츠 밀도가 적절한가요?
   - 섹션 간 구분이 명확한가요?

5. **인터랙션**
   - 호버 효과가 적절한가요?
   - 캐러셀 UX가 직관적인가요?

6. **브랜딩**
   - 로고와 전체 디자인의 조화가 좋은가요?
   - "장소와 취향이 발견되는 공간"이라는 컨셉이 디자인에 잘 반영되어 있나요?

## 참고 사항
- 이 프로젝트는 파주 출판단지의 장소와 행사를 아카이브하는 서비스입니다.
- 타겟 사용자는 책과 문화를 즐기는 20-40대입니다.
- 모바일 반응형 디자인이 필요합니다. (현재는 데스크톱 우선)
