# Story Atlas 전체 탭 디자인 참고 파일

Story Atlas 프론트엔드의 모든 탭(페이지) 디자인 조언을 위한 종합 파일입니다.

## 프로젝트 정보
- **프레임워크**: Next.js 16.0.1 (App Router)
- **스타일링**: Tailwind CSS 4 + CSS Modules
- **폰트**: Pretendard (메인), Dancing Script (필기체), KoPub 바탕체, 미생체, BM 을지로체
- **타겟 사용자**: 책과 문화를 즐기는 20-40대
- **서비스 컨셉**: 파주 출판단지의 장소와 행사를 아카이브하는 서비스

---

## 1. 홈 페이지 (탐색) - `/`

### 개요
메인 페이지로, 히어로 섹션과 카테고리별 콘텐츠 캐러셀을 제공합니다.

### 주요 컴포넌트

#### HeroSection
- **위치**: 페이지 최상단
- **디자인**:
  - 배경: 그라디언트 (`from-[hsl(var(--warm-bg))] to-white`)
  - 장식 요소: 블러 효과가 있는 원형 배경 요소들
  - 배지: "파주 출판단지 아카이브" 배지 (흰색 배경, 그림자)
- **콘텐츠**:
  - 메인 타이틀: "소중한 순간이 기록되는 공간" (일부 텍스트는 브라운 색상 강조)
  - 서브 타이틀: "책과 문화가 살아 숨 쉬는 파주 출판단지의 특별한 장소들을 발견하고, 당신만의 이야기를 만들어보세요."
- **레이아웃**: 중앙 정렬, 최대 너비 7xl

#### CategoryCarousel (7개 카테고리)
각 카테고리는 가로 스크롤 캐러셀로 표시됩니다:

1. **행사** (`events`)
   - 설명: "어, 이번 주말에 뭐 한다고?"
   - 타입: events
   - 데이터: API에서 가져옴

2. **북카페** (`book-cafe`)
   - 설명: "커피는 핑계일 뿐, 활자에 중독되고 싶어"
   - 타입: places
   - 외부 링크: 네이버 지도 링크

3. **브런치** (`brunch`)
   - 설명: "아침과 점심 사이, 그 완벽한 행복의 순간"
   - 타입: places
   - 외부 링크: 네이버 지도 링크

4. **야외카페** (`outdoor-cafe`)
   - 설명: "일단 나와! 커피는 맑은 공기 마시면서"
   - 타입: places
   - 외부 링크: 네이버 지도 링크

5. **대형카페** (`large-cafe`)
   - 설명: "넓은 공간에서 펼치는 나만의 시간"
   - 타입: places
   - 외부 링크: 네이버 지도 링크

6. **주변 관광지** (`tourist-spots`)
   - 설명: "커피 배 채웠으면, 이제 감성 배 채울 시간"
   - 타입: places
   - 데이터: 빈 배열

7. **관광지** (`spots`)
   - 설명: "파주 출판도시의 특별한 장소들"
   - 타입: spots
   - 외부 링크: 네이버 지도 링크

#### CategoryCarousel 컴포넌트 특징
- **카드 크기**: 288px 고정 너비
- **카드 간격**: 1rem (space-x-4)
- **스크롤**: 가로 스크롤 (드래그 지원)
- **네비게이션**: 좌우 화살표 버튼
- **카드 타입**:
  - PlaceCard: 288x192 이미지 + 장소명 + AI 요약 헤드라인
  - EventCard: 288x320 이미지 + 행사명 + 헤드라인 + 기간 (종료된 행사는 배지 + 반투명 처리)

### 레이아웃
- 최대 너비: 7xl (max-w-7xl)
- 좌우 패딩: px-4 sm:px-6 lg:px-8
- 배경: `bg-[hsl(var(--warm-bg))]`

## 2. 행사 페이지 - `/events`

### 개요
모든 행사를 검색, 필터링, 정렬할 수 있는 페이지입니다.

### 주요 기능

#### 헤더 섹션
- **배경**: 그라디언트 (`from-[hsl(var(--accent-brown))]`)
- **제목**: "행사" (흰색, 4xl, font-pretendard-bold)
- **설명**: "파주 출판단지에서 열리는 다양한 문화 행사를 만나보세요" (흰색, text-lg)

#### 검색 및 필터 바
- **검색 입력**: 검색 아이콘 포함, 플레이스홀더 "행사 검색..."
- **정렬 선택**: 드롭다운 (날짜순/이름순)
- **스타일**: 흰색 배경, 둥근 모서리

#### 탭 시스템
1. **진행 중** (`ongoing`)
   - `status !== -1`인 행사만 표시
   - 카운트 표시: "진행 중 (N)"

2. **종료됨** (`ended`)
   - `status === -1`인 행사만 표시
   - 카운트 표시: "종료됨 (N)"

3. **전체** (`all`)
   - 모든 행사 표시
   - 카운트 표시: "전체 (N)"

#### 그리드 레이아웃
- **반응형**: 
  - 모바일: 1열
  - sm: 2열
  - lg: 3열
  - xl: 4열
- **간격**: gap-6
- **카드**: EventCard 컴포넌트 사용

#### EventCard 특징
- **크기**: 288x320 (세로형 포스터)
- **내용**: 이미지 + 행사명 + 헤드라인 + 기간
- **종료된 행사**: 배지 + 반투명 처리
- **호버 효과**: scale-105 (5% 확대)

### 레이아웃
- 최대 너비: 7xl
- 배경: `bg-[hsl(var(--warm-bg))]`
- 하단 패딩: pb-24

## 3. 장소 페이지 - `/places`

### 개요
모든 장소(카페, 관광지)를 카테고리별로 필터링하고 검색할 수 있는 페이지입니다.

### 주요 기능

#### 헤더 섹션
- **배경**: 그라디언트 (`from-[hsl(var(--accent-brown))]`)
- **제목**: "장소" (흰색, 4xl, font-pretendard-bold)
- **설명**: "파주 출판단지의 특별한 장소들을 탐색해보세요" (흰색, text-lg)

#### 검색 바
- **검색 입력**: 검색 아이콘 포함, 플레이스홀더 "장소 검색..."
- **스타일**: 흰색 배경

#### 카테고리 필터
- **카테고리**: 전체, 북카페, 브런치, 야외카페, 대형카페, 관광지
- **표시**: Badge 컴포넌트 사용
- **선택된 카테고리**: 브라운 배경 (`bg-[hsl(var(--accent-brown))]`)
- **스크롤**: 가로 스크롤 가능 (모바일)

#### 뷰 모드 토글
- **그리드 뷰**: 3x3 아이콘
- **리스트 뷰**: 리스트 아이콘
- **토글**: ToggleGroup 컴포넌트 사용

#### 그리드 뷰
- **반응형**: 
  - 모바일: 1열
  - sm: 2열
  - lg: 3열
  - xl: 4열
- **간격**: gap-6
- **카드**: PlaceCard 컴포넌트 사용

#### 리스트 뷰
- **카드 스타일**: 흰색 배경, 둥근 모서리 (rounded-2xl), 그림자
- **레이아웃**: 이미지(128x128) + 텍스트 (가로 배치)
- **호버 효과**: 그림자 확대 (shadow-md → shadow-xl)
- **이미지 호버**: scale-110

#### PlaceCard 특징
- **크기**: 288x192 (가로형)
- **내용**: 이미지 + 장소명 + AI 요약 헤드라인
- **호버 효과**: scale-105 (5% 확대)

#### 결과 카운트
- **표시**: "N개의 장소" (text-sm, text-gray-600)

### 레이아웃
- 최대 너비: 7xl
- 배경: `bg-[hsl(var(--warm-bg))]`
- 하단 패딩: pb-24

## 4. 책갈피 만들기 페이지 - `/bookmark`

### 개요
사용자가 좋아하는 책 구절로 책갈피를 만들 수 있는 폼 페이지입니다.

### 주요 특징

#### 배경 디자인
- **그라디언트**: `from-amber-50 via-orange-50 to-rose-50`
- **장식 요소**:
  - 떠다니는 책 아이콘들 (애니메이션: float, float-delayed, float-slow)
  - 그라디언트 원형 블러 효과 (blob 애니메이션)
  - 색상: amber, orange, rose 계열

#### 헤더 섹션
- **아이콘**: 책갈피 SVG 아이콘 (orange-600)
- **제목**: "나만의 책갈피 만들기" (text-3xl, font-bold)
- **설명**: "좋아하는 책 구절로 세상에 하나뿐인 책갈피를 만들어보세요" (text-gray-600)

#### BookmarkForm 컴포넌트
- **최대 너비**: max-w-2xl
- **중앙 정렬**: mx-auto
- **기능**:
  - 텍스트 입력 (책 구절)
  - 폰트 선택
  - 색상 선택
  - 미리보기
  - 다운로드

### 레이아웃
- 전체 배경: 그라디언트
- 컨테이너: mx-auto px-4 py-8 md:py-12
- z-index: relative z-10 (콘텐츠가 장식 요소 위에 표시)

## 5. 행사 상세 페이지 - `/events/[id]`

### 개요
개별 행사의 상세 정보를 보여주는 페이지입니다.

### 주요 구성

#### 이미지 영역
- **크기**: 
  - 모바일: h-80
  - sm: h-96
  - md: h-[500px]
- **스타일**: object-contain, bg-gray-100
- **종료 배지**: 우측 상단에 "종료" 배지 (status === -1일 때)

#### 콘텐츠 영역
- **제목**: text-4xl, font-extrabold, font-title
- **날짜 정보**:
  - 시작일과 종료일이 같으면 한 줄 표시
  - 다르면 "시작일 ~ 종료일" 형식
  - 장소 정보 (place_id)
  - 주최 정보 (host)
- **헤드라인**: 
  - text-2xl, font-bold
  - 왼쪽 테두리 (border-l-4 border-blue-500)
  - 패딩 (pl-4)
- **본문**: 
  - paragraph1, paragraph2, paragraph3
  - text-lg, leading-relaxed
  - 단락 간 간격: mt-6

### 레이아웃
- 최대 너비: max-w-4xl
- 중앙 정렬: mx-auto
- 패딩: p-6 md:p-10

## 공통 디자인 요소

### 헤더 (Header)
- **위치**: 모든 페이지 상단
- **스타일**: 
  - sticky (홈 페이지만), 일반 (다른 페이지)
  - 배경: bg-white/80, backdrop-blur-md
  - 하단 테두리: border-b
- **구성**:
  - 로고: Story Atlas (BookOpenIcon + 텍스트)
  - 네비게이션: 탐색, 행사, 장소 (데스크톱만 표시)
  - 액션: 책갈피 만들기 버튼

### 색상 팔레트
- **메인 브라운**: `hsl(var(--accent-brown))`
- **웜 배경**: `hsl(var(--warm-bg))`
- **웜 보조**: `hsl(var(--warm-secondary))`
- **액센트 베이지**: `hsl(var(--accent-beige))`
- **텍스트**: 
  - 메인: #000000 (검은색)
  - 서브: #6B7280 (gray-600)
  - 흰색: #FFFFFF

### 타이포그래피
- **제목**: Pretendard Bold (700), font-title
- **본문**: Pretendard Medium (500)
- **로고**: Pretendard ExtraBold (900), 소문자
- **폰트 클래스**:
  - `.font-title`: 제목용
  - `.font-pretendard`: 본문용
  - `.font-dancing-script`: 필기체
  - `.font-kopub-batang`: 바탕체
  - `.font-misengchae`: 미생체
  - `.font-bm-euljiro`: BM 을지로체

### 카드 디자인
- **그림자**: shadow-lg
- **둥근 모서리**: 없음 (직각) 또는 rounded-2xl (일부)
- **호버 효과**: scale-105 (5% 확대)
- **전환**: transition-all duration-300

### 버튼 및 입력
- **버튼**: Button 컴포넌트 (outline, default variants)
- **입력**: Input 컴포넌트 (흰색 배경, 둥근 모서리)
- **배지**: Badge 컴포넌트 (outline, default variants)

### 애니메이션
- **float**: 떠다니는 효과
- **blob**: 블러 원형 효과
- **전환**: transition-all, duration-300

## 디자인 개선 제안 요청 사항

### 전체적인 비주얼
1. **일관성**
   - 각 탭 간 디자인 일관성이 적절한가요?
   - 색상 사용이 통일되어 있나요?
   - 타이포그래피 계층 구조가 명확한가요?

2. **사용자 경험**
   - 네비게이션이 직관적인가요?
   - 검색 및 필터 기능이 사용하기 쉬운가요?
   - 모바일 반응형이 적절한가요?

### 홈 페이지
1. **히어로 섹션**
   - 메시지가 명확하고 매력적인가요?
   - 배경 디자인이 적절한가요?

2. **캐러셀**
   - 카드 크기와 비율이 적절한가요?
   - 스크롤 UX가 직관적인가요?
   - 카테고리 설명 문구가 효과적인가요?

### 행사 페이지
1. **탭 시스템**
   - 진행 중/종료됨/전체 구분이 명확한가요?
   - 카운트 표시가 유용한가요?

2. **검색 및 필터**
   - 검색 기능이 충분한가요?
   - 정렬 옵션이 적절한가요?

### 장소 페이지
1. **카테고리 필터**
   - 카테고리 구분이 명확한가요?
   - 필터 UI가 사용하기 쉬운가요?

2. **뷰 모드**
   - 그리드/리스트 뷰 전환이 자연스러운가요?
   - 각 뷰 모드가 적절한가요?

### 책갈피 만들기 페이지
1. **배경 디자인**
   - 장식 요소가 과하지 않은가요?
   - 폼에 집중할 수 있는 디자인인가요?

2. **폼 UX**
   - 입력 과정이 직관적인가요?
   - 미리보기가 충분한가요?

### 상세 페이지
1. **레이아웃**
   - 이미지와 텍스트 비율이 적절한가요?
   - 정보 계층 구조가 명확한가요?

2. **가독성**
   - 본문 텍스트가 읽기 쉬운가요?
   - 여백이 적절한가요?

### 공통 요소
1. **헤더**
   - 네비게이션이 명확한가요?
   - 로고와 브랜딩이 일관되나요?

2. **색상 및 타이포그래피**
   - 색상 대비가 충분한가요?
   - 폰트 선택이 적절한가요?
